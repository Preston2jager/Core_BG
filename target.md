# 要求

## 总览
1. 这个是一个模拟太空中的能量内核的背景程序，随着本机CPU负荷的上升，能量内核会相应的改变视觉状态。

## 技术要求
1. **渲染架构**：完全交给 GPU 渲染，采用现代 **DirectComposition (DComp)** 视觉树架构，确保窗口与系统桌面背景完美集成。
2. **桌面集成**：程序窗口作为桌面背景的子窗口运行，不干扰桌面图标和用户交互。
3. **原生清晰度**：实现 **1:1 像素对齐渲染**，彻底移除分辨率上限。在 2K、4K 及更高分辨率屏幕上保持点对点的极高清晰度。
4. **系统托盘**：通过系统托盘图标管理（Resume/Pause, FPS Limit, Glow Intensity, Refresh Wallpaper, Exit）。
5. **平滑过渡**：每秒读取一次 CPU 负载，核心状态（颜色、速度、半径）通过插值算法实现极其平滑的视觉过渡。

## 背景
1. **系统壁纸同步**：背景不再使用虚拟星空，而是直接读取并渲染 Windows 系统当前的**原生桌面壁纸**。
2. **无损画质**：以原图尺寸加载（Lossless Loading），不进行任何有损压缩。
3. **高保真采样**：应用 **16x 各向异性过滤 (Anisotropic Filtering)**，确保壁纸在大尺寸缩放时依然锐利。
4. **覆盖式填充 (Cover)**：自动计算壁纸与屏幕的比例，实现“等比例填充并中心裁剪”，确保在任何显示器比例下都不会产生拉伸变形。
5. **多显示器跨屏映射 (Span)**：支持多显示器坐标自动对齐。壁纸会作为一幅完整的画卷跨越所有显示器，每个屏幕渲染其对应的局部片段。

## 前景
1. **能量小球**：24 个发光能量球分布在 3D 空间的球形轨道上。
2. **引力模拟**：小球被屏幕中心的一个“虚拟大质量恒星”引力牵引，进行高速旋转运动。
3. **动态拖尾**：小球带有高频插值的火焰状拖尾粒子。利用 0.1s~0.25s 的极短寿命和随机湍流，营造出能量喷涌的质感。

## 基于荷载动态变化
1. **低荷载**：旋转速度极慢，轨道半径收缩，颜色呈现纯净的白色。
2. **高荷载**：速度大幅提升，半径扩张。颜色向亮橙红色过渡（采用 Pantone 17-1463 亮橙红）。
3. **能量核心 (40%+ 负载)**：
    - 屏幕中心逐渐浮现纯白色辉光**野猪头 Logo**（使用 `logo_t.png`）。
    - 负载越高，Logo 直径越大（100% 负载时约 89.18 像素）。
    - Logo 表面产生高频振动效果，模拟能量不稳定的状态。
    - 采用平方根非线性透明度曲线，确保在中等负载下核心即清晰可见。
4. **静电放电 (50%+ 负载)**：核心向四周随机发射细小的**电蓝色静电闪电粒子**。这些粒子以折线形式瞬间闪现并消失。
5. **粒子上限**：渲染管线支持高达 15,000 个实例，确保在高负载下特效依然流畅丰富。
